    Отчет по лабораторной работе №3

    Выполнил Скобликов Кирилл, K3339

    Library API

    Бэкенд‑API для управления библиотекой: книги, читатели, залы, выдачи, статистика и отчёты.

    Авторизация

    Все эндпоинты требуют авторизации (IsAuthenticated).

    # Регистрация
POST /auth/users/
{
  "username": "user",
  "password": "password",
  "re_password": "password"
}

# Логин
POST /auth/token/login/
{
  "username": "user",
  "password": "password"
}
→ {"auth_token": "<token>"}

    В дальнейшем токен передаётся в заголовке:

    Authorization: Token <token>

    Книги

    Получить список книг

    GET /books/

    Создать книгу

    POST /books/
{
  "book_name": "1984",
  "authors": "George Orwell",
  "publishing_house": "Secker & Warburg",
  "publication_year": "1949-01-01",
  "cipher": "A-123"
}

    Получить книгу по ID

    GET /books/{id}/

    Изменить книгу

    PATCH /books/{id}/
{
  "cipher": "B-777"
}

    Удалить книгу

    DELETE /books/{id}/

    Читатели

    Получить список читателей

    GET /readers/

    Создать читателя

    POST /readers/
{
  "first_name": "Иван",
  "last_name": "Иванов",
  "reader_card_number": "RC123456",
  "birth_date": "2003-05-12",
  "passport_number": "123456",
  "phone_number": "+79991234567",
  "education": "he",
  "academic_degree": false,
  "address": "Москва",
  "hall": 1
}

    Изменить читателя

    PATCH /readers/{id}/

    Удалить читателя

    DELETE /readers/{id}/

    Книги у читателя

    GET /readers/{id}/books/

    Чтение (выдача книг)

    Получить все выдачи

    GET /readings/

    Выдать книгу

    POST /readings/
{
  "book": 1,
  "reader": 2,
  "issued_date": "2025-01-10"
}

    Получить выдачу

    GET /readings/{id}/

    Изменить выдачу

    PATCH /readings/{id}/
{
  "returned_date": "2025-02-01"
}

    Удалить выдачу

    DELETE /readings/{id}/

    Залы

    Получить список залов

    GET /halls/

    Создать зал

    POST /halls/
{
  "hall_number": 1,
  "hall_name": "Главный зал",
  "capacity": 50
}

    Получить зал

    GET /halls/{id}/

    Изменить зал

    PATCH /halls/{id}/

    Удалить зал

    DELETE /halls/{id}/

    Статистика

    По уровню образования

    GET /readers/stats/education/

    Читатели младше 20 лет

    GET /readers/stats/young/

    Плохие читатели (более месяца)

    GET /readers/bad/

    Читатели с редкими книгами

    GET /readers/rare-books/

    Отчёты

    Ежемесячный отчёт библиотеки

    GET /reports/monthly/{year}/{month}/

    Сотрудники

    Юзернеймы сотрудников

    GET /library-employees/
